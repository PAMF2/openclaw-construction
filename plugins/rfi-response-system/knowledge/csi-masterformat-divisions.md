# CSI MasterFormat Divisions — Complete Reference

> Construction Specifications Institute (CSI) MasterFormat 2018 Edition
> Used for organizing construction specifications, cost data, and project documentation.

| 00 80 00 | Unassigned | — |

**Common use:** Front-end documents in every project manual. AIA A201 General Conditions fall under 00 72 00.

## Notes

- **MasterFormat 2018** is the current edition. Section numbers are organized in a **XX YY ZZ** format: Division, Section Level 2, Section Level 3.
- Divisions 15 and 16 (legacy mechanical and electrical) were replaced by Divisions 21-28 in MasterFormat 2004. Some older projects still reference 15 and 16.
- Not all section numbers are actively used on every project. The project specification table of contents defines which sections apply.
- For RFI responses, always reference the specific section number (e.g., "per Section 07 92 00, paragraph 2.3.A") rather than generic division references.
